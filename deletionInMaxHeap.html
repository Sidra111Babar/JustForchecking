Deletion in Max Heap

  1. Identify the root:In a Max-Heap, the element to be deleted is typically the root (maximum element), as the heap maintains the largest element at the root.
  2. Replace the root:Replace the root element with the last element of the heap (the rightmost leaf).
  3. Remove the last element: After replacing the root, the max element is no longer needed, so remove it.
