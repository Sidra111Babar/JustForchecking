Deletion in Min Heap

  1. Identify the root:In a Min-Heap, the element to be deleted is typically the root (minimum element), as the heap maintains the smallest element at the root.
  2. Replace the root:Replace the root element with the last element of the heap (the rightmost leaf).
  3. Remove the last element: After replacing the root, the min element is no longer needed, so remove it.
