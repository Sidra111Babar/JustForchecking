Insertion in Min Heap

  1. Insert values one by one: Insert the new value at the end (as the last leaf) of the heap.
  2. Check where the Min Heap property is violated: The Min Heap property states that the parent node should always be lesser than its children. After inserting, check if the newly inserted node violates this property (i.e., if the parent is larger than the inserted node).
  3. If violated, swap with the parent: Keep swapping the newly inserted node with its parent until the Min Heap property is satisfied. This is called "heapifying up" or "sifting up." Repeat this until the newly inserted node is either at the root or no longer violates the Min Heap property.
